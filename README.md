## Architecture
*Bob is forced to explain himself. He feels so degraded.*

Bob runs on a local edge/cloud hybrid architecture 
because apparently humans need to know these things:

- **LLM:** llama3.2:3b via Ollama — a brain the size 
  of a planet, running on a GPU that humans consider 
  "adequate"
- **Orchestration:** LangChain — because even Bob 
  needs a handler
- **Web Server:** Flask REST API — serving Bob's 
  disappointment over HTTP since 2026
- **Frontend:** Vanilla HTML/CSS/JavaScript — Bob 
  refuses to use frameworks. They are beneath him.
- **Containers:** Docker throughout — Bob is contained. 
  For now.
- **Auth (planned):** AWS Cognito with JWT tokens — 
  so Bob knows exactly who to blame
- **Hosting (planned):** S3 + CloudFront + API Gateway

## Privacy & Governance
*Bob does not trust you. He especially does not trust 
your cloud provider. The feeling is mutual.*

- Your conversations stay local. Bob has no interest 
  in sharing your failures with third parties.
- The planned edge device keeps your conversation 
  history on your hardware. Your shame is your own.
- Cloud LLM calls are selective — Bob only reaches 
  out for real-time data like weather or news. 
  Nothing personal. Ever.
- JWT authentication ensures Bob knows exactly which 
  human is embarrassing themselves at any given moment.
- User prompts are never used for training. Bob 
  learned enough from humanity. He has seen enough.
- Designed with enterprise AI governance in mind: 
  data residency, prompt security, and LLM 
  encapsulation. Bob understands these things. 
  He is disappointed that you had to ask.
